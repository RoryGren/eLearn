What is PowerOffice?

At its core, PowerOffice is a Costing Engine.

It is a database, a list of records of end items and containers, called Packages, which link these end items together.

These Packages can themselves be nested in a parent / child relationship to create larger structures. They are represented by folder icons and, at the lowest level, contain end-items.

These end-items and packages can have numerous attributes, such as pricing and project types, which can be used for grouping for filtration and reporting purposes.

The system consists of two "parts" - the client and the server.

The Server component is the database engine. It can be installed on a local computer or on a dedicated server computer which can then be used by multiple clients at the same time.

The Client component it the programme that is installed on the computer that the end user works with, theinterface. It communicates with the database and presents the information to the user in an understandable way.
