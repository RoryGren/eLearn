Filtering with "Wild Cards"

There are two Wild Card symbols that can be used as part of the search query with the PowerOffice database:

  • % - This is the multiple character wild card. It will allow itself to be replaced by multiple characters.
  • _ - This is the single character wild card. It will allow itself to be replaced by a single character - one character only. This option can be used multiple times to represent a specific number of characters in the search.

Note: The multiple character Wild Card is built in to the Advanced Filter. You do NOT need to add it into your search query.
