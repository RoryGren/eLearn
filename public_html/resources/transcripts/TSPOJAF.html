Using the Advanced Filter

The "Advanced" Filter as a little more sophisticated than the Quick Filter, in that, over and above filtering on the Description, Code and Alt Code columns, it will also filter on Attributes.

The Advanced Filter is started by clicking on the ellipsis button. (The square button with three dots '...') This opens up the Advanced Filter dialog box, which contains the following fields:

  • On the left hand side, the Attribute drop down box.
  • In the centre, the Operator.
  • On the right hand side, the Value.

Depending on the Attribute selected,

  • the Operator will have different values available,
  • and the Value will either have values in a drop-down list, or be an empty field which can have custom values typed into it.
Buttons

Clicking the PLUS button adds a row to the filter criteria for additional filter options.

Clicking the Delete button to the left of the PLUS button, removes that row from the filter criteria.

Clicking the OK button activates the filter, closing the dialog box and putting a tick in the Filter check box to show that the Advanced Filter is active. Removing the tick by clicking it, de-activates the filter and returns PowerOffice

Clicking the X or Cancel button to the right of the OK button, cancels the current filter operation.

Notes:

  1. PowerOffice will remember the last used advanced filter AND Quick Filter settings. Simply putting a tick into the Filter checkbox will re-activate it. PowerOffice will show the remembered search criterea to the right of the ellipsis button.
  2. The Advanced Filter and the Quick Filter work in conjunction with each other. I.e. The result set returned by the one will be further filtered by the other.
