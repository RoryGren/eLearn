Projects and Versions

Projects

  • In PowerOffice, a Project is represented by a yellow folder icon.
  • Double clicking a Project opens it up to reveal its contents, be it sub-projects, sub-sub-projects or versions and its icon changes to an open folder.
  • A Project is a container, which can contain sub-projects or child-projects and Versions. It holds everything together.
  • A Project can only contain sub-projects and Versions.

Versions

  • In PowerOffice, a Version is represented by an icon of a blue book.
  • Double clicking a Version opens the version to reveal its contents, which will be packages, or structures, and end items - the actual BOM of your project, and its icon changes to an open book.
  • The actual "nuts and bolts" of your project are contained in a Version.
  • The Version could be a "version" of the project, a stage or a phase of the project, a subset of the project, such as one transformer area inside a township with multiple transformer areas.
  • A Version can only contain Packages and Items.
  • A Version can only be contained inside a Project.
