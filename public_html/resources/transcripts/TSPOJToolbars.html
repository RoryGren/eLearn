PowerOffice Projects Toolbars.

Toolbars are a row of "buttons" below the menu bar.
They are shortcuts which usually perform actions on the entire screen, or on highlighted or selected records in the screen below them.
Buttons sometimes have a drop-down arrow on their right, indicating that there is a sub-menu, with further selections available to the user, accessed by clicking the drop-down arrow.
Buttons can sometimes be "grayed out," or inactive, indicating that the action they would perform cannot be performed on the highlighted, or selected, record in the main screen below them.

In the Projects module the following buttons are available on the left-hand side of the screen:

  • Project: This button has a drop-down arrow with the following:
    • Project: which is used to create new projects.
    • Version: Create a new project version inside the highlighted project.
    • User Item: Create a User Item, which is a "buy-out" item, I.e. an item which is NOT in the database, but the cost of which, still needs to be added to the project. This User Item will NOT be sent through to SAP with the Material Reservation.
  • Cost: This button costs the project, or highlighted record and its children, according to the selections made when the version is created.
    The drop-down for this button allows the user to select Quick Cost, which will be discussed later in the course.
  • App Button: This button allows access to various additional applications which become available to the user, depending on the selection in the main screen:
    • ElecTech: a pre-quoting, pre-planning, pre-designing, technology "suggestion" tool,
    • Elec: A design interface between Bentley Microstation CAD, PowerOffice and ReticMaster electrical analysis tool,
    • Subs: A design interface between Bentley Microstation CAD and PowerOffice,
    • Lines: A design interface between Bentley PLSCADD and PowerOffice,
    • PMIS: an interface between PMIS, a bespoke management application, and PowerOffice
  • Report: Opens the PowerOffice Reports module.
  • Wizard: Opens the PowerOffice Wizard module.
  • Refresh: re-reads the PowerOffice database.
  • Help: Opens the PowerOffice Help file.

On the right hand side, there are the Refresh button and the Next button.
