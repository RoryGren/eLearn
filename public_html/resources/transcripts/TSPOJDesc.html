Descriptions and Comments

Descriptions
  • Can be changed from the default on Packages, and NOT end-items, by right clicking the relevant package and selecting Edit.
  • In the previous section we came to this dialog box to change the quantity and we can see here that the Description can also be changed and it can be changed to anything.
  • Changing the description here only affects the selected package in the selected project, in the selected version. It does not affect the templates or any other projects or versions.
Comments
  • Can be added to Packages and End-Items.
  • Can be added and modified by right clicking the relevant package or end-item and selecting Comments.
  • The comment is not limited by length and can be used for notes.
Where Is Used

In the right hand screen, right clicking any package or end item and selecting Where Is Used opens a dialog box which will show where the selected package or end-item is used in PowerOffice,

  • in other packages in the templates,
  • in any projects, and
  • in and Wizards.
Show Node Data as Text

There is NO copy and paste function on the templates on the right hand side of the screen as everything is Read Only.

In order to be make searching easier, right clicking the selected package or item on the templates screen, allows Show Node Data As Text.

This option reveals all the information about the selected record in a text box where text can be selected and copied for whatever reason the user requires.
